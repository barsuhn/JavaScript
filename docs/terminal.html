Navigation im Dateisystem

Eine kurze Liste von Terminal-Kommandos für die Navigation im Dateisystem:

ls - list
Liste die Dateien im aktuellen Verzeichnis auf.
  • -l (long) Ausführliches Listing mit zusätzlichen Dateiinformationen
  • -a (all) Listet auch die unsichtbaren Dateien und Verzeichnisse auf
pwd - print working directory
Zeige das aktuelle Arbeitsverzeichnis an.
cd - change working directory
Wechsle in ein anderes Arbeitsverzeichnis.
mkdir - make directory
Erzeuge ein neues Verzeichnis.
rmdir - remove directory
Lösche ein leeres Verzeichnis.
touch - change file access and modification times
Ändere die Zugriffs und Änderungszeit einer Datei. Falls die Datei nicht existiert, wird sie erzeugt.
rm - remove
Lösche eine oder mehrere Dateien.
  • -r (recursive) Lösche alle enthaltenen Dateien und Unterverzeichnisse.
